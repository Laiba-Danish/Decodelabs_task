# Decodelabs Data Analytics Internship

## Overview
This repository contains the tasks completed during the Decorelabs Data Analytics Internship. The projects demonstrate core data analytics skills, including data cleaning, exploratory data analysis (EDA) and SQL-based data analysis using Microsoft Excel and MySQL.

---

## Task 1: Data Cleaning & Preparation

### Description
Prepared a raw dataset for analysis by improving its quality and consistency.

### Key Activities
- Cleaned the dataset by handling missing and inconsistent data.
- Applied filters to organize and inspect the data.
- Prepared the dataset for further analysis.

*Tools Used:* Microsoft Excel

---

## Task 2: Exploratory Data Analysis (EDA)

### Description
Performed exploratory data analysis to identify patterns and generate meaningful insights.

### Key Activities
- Calculated descriptive statistics (Mean, Median, Count, Maximum, and Minimum).
- Created Pivot Tables to summarize data.
- Analyzed trends and patterns within the dataset.
- Summarized key findings from the analysis.

*Tools Used:* Microsoft Excel

---

## Task 3: SQL Data Analysis

### Description
Used SQL to retrieve, filter, and analyze data from a relational database.

### Key Activities
- Created and managed a MySQL database.
- Imported the dataset into MySQL.
- Executed SQL queries to retrieve and analyze data.
- Used SQL clauses including SELECT, WHERE, GROUP BY, ORDER BY, HAVING, and aggregate functions.
- Extracted meaningful insights from the data.

*Tools Used:* MySQL Workbench, SQL
